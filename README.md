## Features
- Add, edit, and delete songs in the music library
- Store song metadata (Title, Artist, Album, Genre, Release Year, Duration, etc.)
- MySQL database integration for song storage
- Command-line interface (CLI) for interaction

### Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/yourusername/music-library-project.git
   cd music-library-project

2. Create a Virtual Environment
    python -m venv venv
    source venv/bin/activate  # On Windows use 'venv\\Scripts\\activate'

3. Install dependecies
    pip install -r requirements.txt

4. Set Up MySQL
    python setup.py

5. Run the app
    python app.py

6. Click the link
    Running on http://127.0.0.1:5000
